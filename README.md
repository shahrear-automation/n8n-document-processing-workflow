## 🛠 Requirements

- n8n (Self-hosted or Cloud)
- Google Drive API credentials
- OpenAI / Gemini API (for AI extraction)
- OCR service (e.g., OCR.space or Google Vision)

---

## 🚀 Setup Instructions

1. Import the provided `.json` workflow into n8n  
2. Configure credentials:
   - Google Drive
   - Gmail (optional)
   - AI API (OpenAI / Gemini)
3. Update folder IDs and API keys  
4. Activate the workflow  

---

## ⚠️ Notes

- Some low-quality or blurry images may fail OCR  
- Free OCR services may have rate limits  
- AI extraction accuracy depends on document clarity
